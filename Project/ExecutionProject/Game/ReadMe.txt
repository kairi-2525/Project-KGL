シーン一覧
    ロードシーン
        (ShaderToyを使用し、リッチな画面に仕上げました。)
    花火エディターシーン
        （一番見てほしいシーン）

操作説明
    WASD
        移動
    マウス右クリックしながらマウスを動かす
        視点移動
    マウス左クリック
        選択中のパーティクルをカメラから発射
        （発射したパーティクルに他のパーティクルが吸い寄せられる）
    F1
        UI非表示
    Ctrl + Enter
        シーンリセット
    Esc
        終了

GUI解説
    左側
        タイムスケールを変更
        パーティクルの生成フラグの変更やパーティクルの削除
            パーティクル生成フラグはSpawner項目を選択してから生成されます。
            Spawner項目については後述
        各項目のDebug用パラメーター
    右側
        花火のデモを再生できます
    中央
        各種項目からエディターウィンドウを表示します。
        Sky
            スカイボックスを変更します。
        Particle
            パーティクルエディタを表示します。
        Spawner
            一覧から選択し、選択中のSpawnerからパーティクルを生成することができます。
        Render Target
            レンダーターゲットを確認できます。
        Options
            様々な値を確認できます。
        Debug  
            キューブを表示します（アンチエイリアスの確認に使用しました。）
